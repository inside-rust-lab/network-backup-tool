Project: Network Device Backup & Inventory Tool

Build a CLI tool that:

Reads a list of network devices from a file
Connects to them (simulated at first)
"Retrieves" their configuration
Saves backups
Logs success/failure
Handles connection errors
Later you can swap the mock connection with Netmiko/NAPALM/Paramiko.

Create a README.md

Logging:

Create logging in:

logs/backup.log
Use Python’s logging module.
Log things like:

INFO - connecting to core-sw1
SUCCESS - backup complete
ERROR - connection failed

Concepts:

standard library
log levels
formatting


Error Handling:

Add error cases:

Simulate failures like:

ConnectionError
TimeoutError
AuthenticationError

Handle with:

try
except
finally

Example

try:
    device.connect()
    config = device.get_config()
except ConnectionError:
    log failure

This is critical for automation scripts in real environments.



Add Useful Python Features:

Add these intentionally:

Sets

Track unique vendors:

vendors = set()
Tuples

Return structured results:

return (hostname, status)
List comprehensions

Example:

cisco_devices = [d for d in devices if d.vendor == "cisco"]
